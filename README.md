# 吉法師の野望

1986年の歴史シミュレーションゲームにインスパイアされた、ブラウザで動く戦国シミュレーションゲーム。ビルド不要の素の HTML/CSS/JavaScript（ES2020・classic script）で実装されています。

## 遊び方

```sh
open index.html
```

file:// のままで動作します（fetch 等は不使用）。ローカルサーバーを使う場合:

```sh
python3 -m http.server 8000
# → http://localhost:8000
```

## ゲーム概要

- **モード**: 全国50ヶ国モード / 17ヶ国モードの2種類。1560年春からスタート。
- **大名**: 史実の44家から選択。能力値はスロットで決定。
- **ターン進行**: 季節ごとに「イベント → プレイヤーコマンド → 敵大名の思考」が進む。
  - 季節イベント: 加齢・収入・台風・収穫・疫病・一揆・謀反・本能寺の変など。
- **戦闘**: プレイヤーが関与する戦いは 7x7 グリッドの戦術マップ戦闘（最大5部隊・兵糧と30日制限）。AI 同士や空白地攻めは自動解決。
- **死亡規則（原作準拠）**: 討死→全領土が勝者へ / 病死・暗殺→全領土空白化 / 本国での一揆・謀反→本国に新大名。空白国は民兵が守り、春に新大名が勃興することも。
- **軍師の進言**: 自分のターンに、その国で今取るべき行動を軍師が理由付きで進言。推奨コマンドには⭐バッジが付き、「他の策」で次善の候補も見られる（選ぶとそのまま実行）。出陣先選びでも狙い目の国を教えてくれる。
- **セーブ/ロード**: localStorage に保存。ロードすると当該季節の頭から再開。
- **似顔絵**: 大名の顔は外部画像なしの SVG 自動生成（同じ大名は常に同じ顔）。
